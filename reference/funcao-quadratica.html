Função Quadrática

Folha de referência · Matemática do ENEM · Emelly

A forma

f(x) = ax² + bx + c    com a ≠ 0

Se a = 0, o termo x² some e a função vira uma reta. É o x² que faz a parábola.

Para onde a parábola abre

Sinal de aConcavidadeO vértice é
a > 0para cima 🙂o ponto de mínimo
a < 0para baixo 🙁o ponto de máximo

Truque: o sinal de a é a “boca” da parábola. Positivo sorri, negativo entristece.

O vértice — a fórmula que mais cai

xv = −b / (2a)
yv = f(xv)  (basta substituir)

Existe também y_v = −Δ/(4a), mas substituir x_v na função é mais seguro e dá menos chance de errar sinal.

Ordem importa. Ache xv primeiro; só depois calcule yv. Se a pergunta é “em que instante atinge o ponto mais alto?” → resposta é xv. Se é “qual é a altura máxima?” → resposta é yv. Trocar os dois é o erro mais comum.

Raízes (onde a parábola corta o eixo x)

Δ = b² − 4ac     x = (−b ± √Δ) / (2a)
ΔRaízes reais
Δ > 0duas, distintas
Δ = 0uma (vértice toca o eixo x)
Δ < 0nenhuma

Como reconhecer que é quadrática (e não reta)

Diante de uma tabela de valores, olhe as diferenças entre termos consecutivos:

x1234
y4163664
1ª diferença—122028
2ª diferença——88

1ª diferença constante → função afim (reta). 2ª diferença constante → função quadrática (parábola). Aqui a 2ª diferença é constante (8), logo é quadrática: y = 4x².

Contínuo ou pontilhado? Se a grandeza do eixo x pode assumir qualquer valor (corrente elétrica, tempo, distância), o gráfico é uma curva contínua. Só desenhe pontos soltos se a grandeza for contável (número de pessoas, de caixas).
